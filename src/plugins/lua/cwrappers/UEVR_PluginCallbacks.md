# UEVR_PluginCallbacks

Accessed with `uevr.params.callbacks`

## Members

### `callbacks.on_present`

Called when the swapchain presents a frame.

### `callbacks.on_device_reset`

Called when the VR device is reset.

### `callbacks.on_message`

Called when a Windows message is received.

### `callbacks.on_xinput_get_state`

Called when XInput get state is requested.

### `callbacks.on_xinput_set_state`

Called when XInput set state is requested.

### `callbacks.on_post_render_vr_framework_dx11`

Called after the VR framework renders on DX11.

### `callbacks.on_post_render_vr_framework_dx12`

Called after the VR framework renders on DX12.

### `callbacks.on_custom_event`

Called when a custom event is dispatched.
