# UEVR_RendererData

Accessed with `uevr.params.renderer`

## Members

### `renderer.renderer_type`

The type of renderer (e.g., D3D11, D3D12).

### `renderer.device`

The graphics device handle.

### `renderer.swapchain`

The swapchain handle.

### `renderer.command_queue`

The command queue handle.
