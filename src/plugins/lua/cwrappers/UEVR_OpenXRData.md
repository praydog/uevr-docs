# UEVR_OpenXRData

Accessed with `uevr.params.openxr`

## Members

### `openxr.get_xr_instance()`

Returns the OpenXR instance handle.

### `openxr.get_xr_session()`

Returns the OpenXR session handle.

### `openxr.get_stage_space()`

Returns the OpenXR stage space handle.

### `openxr.get_view_space()`

Returns the OpenXR view space handle.
