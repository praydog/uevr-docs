### `sdk.callbacks`

Returns UObject array functions.

### `sdk.ffield`

Returns FField functions.

### `sdk.fproperty`

Returns FProperty functions.

### `sdk.ustruct`

Returns UStruct functions.

### `sdk.uclass`

Returns UClass functions.

### `sdk.ufunction`

Returns UFunction functions.

### `sdk.uobject_hook`

Returns UObjectHook functions.

### `sdk.ffield_class`

Returns FFieldClass functions.

### `sdk.fname`

Returns FName functions.

### `sdk.console`

Returns console manipulation functions.
