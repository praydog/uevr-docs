# UEVR_FUObjectItem

Represents an item in the global UObject array. Returned by [UEVR_FUObjectArray](FUObjectArray.md) `get_item()`.

## Members

- `object: UEVR_UObject*` - The UObject pointer.
- `flags: int` - Object flags.
- `cluster_index: int` - Cluster index.
- `serial_number: int` - Serial number.
