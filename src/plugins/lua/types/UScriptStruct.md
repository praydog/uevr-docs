# UEVR_UScriptStruct

Represents a script struct (POD data structure) in Unreal Engine. Inherits from [UEVR_UStruct](UStruct.md).

## Functions

### `UEVR_UScriptStruct.static_class()`

Returns the [UEVR_UClass](UClass.md) descriptor for UScriptStruct.

### `struct:get_struct_size()`

Returns the size of the struct in bytes.
